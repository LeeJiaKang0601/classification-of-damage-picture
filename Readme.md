## 4.裁切方式說明

首先確認牆體在照片具體位置，然後裁切讓牆位於照片中心，並且不涵蓋其他雜物，同時保留牆體的邊界涵蓋柱和梁

### 4.1.裁切要求

i. 讓牆體整體位於照片**中心**。

ii. 拍攝時要距離牆體有一定距離，盡可能讓牆體邊界涵蓋柱子和梁

iii. 盡可能讓牆體佔比整張圖片的**50%以上**。

iv. 盡可能去掉避免將其他雜物涵蓋進去。
