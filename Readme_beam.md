## 4.裁切方式說明

首先確認梁在照片具體位置，然後裁切讓梁位於照片中心，並且不涵蓋其他雜物。

### 4.1.裁切要求

i. 讓梁位於照片**中心**。

ii. 拍攝時要距離梁有一定距離，盡可能涵蓋梁左右兩段的**柱子**，同時看的到上下的邊界。

<img src="./梁示意照/A025_20-_20(1).jpg" width="80%" />


iii. 盡可能讓梁佔比整張圖片的**50%以上**。

iv. 盡可能去掉避免將其他雜物涵蓋進去。

#### 5.拍攝建議

i. 讓梁位於照片**中心**。

ii. 拍攝時要距離梁有一定距離，盡可能涵蓋梁左右兩段的**柱子**，同時看的到**梁頂上的天花板和梁底下部分牆體或空隙**作為邊界。

iii. 盡可能讓梁佔比整張圖片的**50%以上**。

iv. 盡可能去掉避免將其他雜物涵蓋進去。
